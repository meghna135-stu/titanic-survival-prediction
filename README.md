# titanic-survival-prediction

# *Project Overview*

Predict whether a passenger survived the Titanic disaster using machine learning techniques.

## **Dataset**

Dataset: Kaggle Titanic Competition dataset
Training samples: 891 passengers

### **Steps Performed**

-Data exploration and understanding

-Missing value handling

-Feature selection

-Categorical encoding

-Feature engineering

-Model training and evaluation

-Data Preprocessing

-Filled missing Age values using the median.

-Filled missing Embarked values using the mode.

-Dropped Cabin because most values were missing.

-Encoded categorical variables into numerical values.

-Feature Engineering

### **Created:**
-FamilySize

-IsAlone

### **Features Used**
-Pclass

-Sex

-Age

-Fare

-Embarked

-FamilySize

-IsAlone

# Model

Random Forest Classifier

## **Results**

**Single validation split accuracy: 77.1%**

**5-fold cross-validation accuracy: 80.9%**
